# PrepSaga

PrepSaga is an AI-powered learning platform that helps users study various technical topics, get personalized feedback, and improve their problem-solving skills. This application focuses on topics related to Java, DSA (Data Structures and Algorithms), Spring Boot, Angular, React, and System Design.

### Features:
- **Topic Breakdown**: Learn key concepts in DSA, Java, Spring Boot, Angular, React, and more.
- **AI Feedback**: Get personalized feedback and suggestions for improvement.
- **Interactive Learning**: Engage with coding challenges, mock interviews, and system design problems.
- **Track Progress**: Keep track of your learning progress and skills development.

### Technologies Used:
- **Frontend**: React, Material-UI, OAuth2 (Google login)
- **Backend**: Spring Boot, JWT Authentication, Spring Security
- **Cloud**: Azure App Services, Azure Blob Storage, Azure SQL Database
- **AI**: Integration with GPT-4 for providing intelligent feedback
- **Deployment**: Azure for scalable cloud infrastructure
- **Version Control**: GitHub for code repository and version tracking
